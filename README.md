# Losant University Resources

This repository contains a collection of resources for Losant University. The repository is sorted by course and chapter.

## Templates

If the file is named `template.zip`, it is an applpication that you can import directly into your Losant account. Here are the steps:

- Clone this repository to your local computer.
- Select the `template.zip` you'd like to use.
- Use [Losant's Application Import](https://docs.losant.com/applications/import-export/#importing-an-application) functionality to import the zipped template as a new application.
